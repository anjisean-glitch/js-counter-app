# JavaScript Counter App

A simple interactive counter built using HTML, CSS, and JavaScript.  
This project was created as part of my learning journey in frontend development.

---

## 🚀 Live Demo

Check out the live version here:  


---

## 📌 Features

- Increase counter value
- Decrease counter value
- Reset counter to zero
- Keyboard support (Arrow Up / Arrow Down)
- Real-time UI updates

---

## 🛠️ Technologies Used

- HTML
- CSS
- JavaScript (Vanilla JS)

---

## 🧠 What I Learned

While building this project, I practiced and improved my understanding of:

- DOM manipulation
- Event listeners in JavaScript
- Updating UI dynamically
- Handling user interactions (buttons + keyboard input)
- Debugging basic JavaScript logic issues
